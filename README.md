📖 Project Overview

This project implements the Bubble Sort algorithm in Java to sort an array of integers. It demonstrates how sorting works step-by-step using comparisons and swaps.

The program displays:

- The original array
- The array sorted in descending order
- The array sorted in ascending order

Key concepts covered:

- Arrays in Java
- Nested loops
- Sorting algorithms
- Method implementation
- Conditional logic

-------------------
🚀 Features

✅ Displays original array
✅ Sorts array in descending order
✅ Sorts array in ascending order
✅ Uses optimized Bubble Sort (early exit if already sorted)
✅ Clear console output

--------------------
🛠️ Technologies Used

- Java
- Standard Java libraries (java.util.Arrays)

---------------------
▶️ How to Run

Using NetBeans:
1. Open NetBeans
2. Load the project
3. Run BubbleSorting.java

-----------------------
🔄 How It Works

1. An array is initialized: int[] arr = {64, 34, 25, 12, 22, 11, 90, 5, 55};
2. The program prints the original array
3. It sorts the array in descending order using Bubble Sort
4. It then sorts the same array in ascending order
5. Results are printed after each operation

🧮 Example Output

Original array: [64, 34, 25, 12, 22, 11, 90, 5, 55]
Sorted array in descending order: [90, 64, 55, 34, 25, 22, 12, 11, 5]
Sorted array in ascending order: [5, 11, 12, 22, 25, 34, 55, 64, 90]

------------------------
⚠️ Limitations

- Fixed array (no user input)
- Not efficient for large datasets (O(n²) complexity)
- Console-based only
